Objetos JavaScript

Um objeto JavaScript é uma coleção de propriedades, e uma propriedade é uma associação enre um nome(ou chave) e um valor. O valor de uma propriedade pode ser uma função, que é então considerada um método do objeto

Os objetos JavaScript são usados para armazenar dados e funcionalidades relacionadas. Eles podem ser criados usando a sintaxe de objeto literal, a uncção construtora ou a classe.

Exemplo de objeto literal:

const pessoa = {
    nome:"João",
    idade:30,
    saudacao function() {
        console.lg("Olá, meu nome é" + this.nome);
    }
};
